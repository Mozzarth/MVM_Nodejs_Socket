## Descripción

Proyecto que captura la modificación de un archivo `valores.txt` y que notifica en tiempo real a traves de una conexión SOCKET 

### Instalar dependencias
> npm i
## Correr proyecto
>- Nota : Para entornos que no sean producción copie el archivo .env.sample en la raiz del proyecto con nombre .env y configure el entorno.

- Dev : Para ejecutar en un entorndo de desarollo ejecutar _`npm run dev`_
